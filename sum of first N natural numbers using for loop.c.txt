Sum of First N Numbers:

Problem Statement
Find the sum of numbers from 1 to N.
Input Format
Integer N
Output Format
Print the sum
Test Cases
Input: 5 → Output: 15
Input: 10 → Output: 55
Input: 1 → Output: 1


#include<stdio.h>
int main(){
    int n,i;
    printf("Input:");
    scanf("%d",&n);
    int sum=0;
    for(i=1;i<=n;i++){
       sum+=i;
        printf("%d ",sum);
    }
}
