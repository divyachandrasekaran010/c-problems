Print quotient and remainder of two integers.

Test Case 1
Input: 9 2
Output: Quotient=4 Remainder=1
Test Case 2
Input: 10 3
Output: Quotient=3 Remainder=1
Test Case 3
Input: 20 4
Output: Quotient=5 Remainder=0

program:
#include<stdio.h>
int main(){
    int a,b;
    printf("Input:");
    scanf("%d %d",&a,&b);
    int c=a/b;
    int d=a%b;
    printf(" quotient:%d remainder:%d",c,d);
    return  0;
}
