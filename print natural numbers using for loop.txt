Problem Statement
Write a program to print numbers from 1 to N.
Input Format
Integer N
Constraints
1 ≤ N ≤ 1000
Output Format
Print numbers from 1 to N (space separated)
Test Cases
Input
5
Output
1 2 3 4 5

#include<stdio.h>
int main(){
    int n,i;
    printf("Input:");
    scanf("%d",&n);
    for(i=1;i<=n;i++){
        printf("%d ",i);
    }
}
