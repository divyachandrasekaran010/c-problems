Even Numbers Printer
Description
Print all even numbers from 1 to N.
Problem Statement
Use a for loop to print even numbers.
Input Format
Integer N
Output Format:
Even numbers separated by space
Test Cases
Input: 10 → Output: 2 4 6 8 10
Input: 5 → Output: 2 4
Input: 2 → Output:

#include<stdio.h>
int main(){
    int n,i;
    printf("Input:");
    scanf("%d",&n);
    for(i=2;i<=n;i+=2){
        printf("%d ",i);
    }
}
