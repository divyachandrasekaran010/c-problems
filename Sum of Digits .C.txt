Sum of Digits
Description
Find the sum of digits of a number.
Problem Statement
Write a program to calculate the sum of all digits in the given number N using a for loop.
Input Format
Integer N
Output Format
Print the sum of digits
Test Cases
Input: 123 → Output: 6
Input: 405 → Output: 9
Input: 9 → Output: 9


#include<stdio.h>
int main(){
    int n,sum=0;
    printf("Input:");
    scanf("%d",&n);
    for(;n!=0;n=n/10){
        sum=sum+(n%10);
    }
    printf("output:%d",sum);
    return 0;
}
