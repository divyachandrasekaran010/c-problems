Reverse a Number
Description
Reverse the digits of a number using a for loop.
Problem Statement
Write a program to reverse the given number N using a for loop.
Input Format
Integer N
Output Format
Print the reversed number
Test Cases
Input: 123 → Output: 321
Input: 100 → Output: 1
Input: 9 → Output: 9


#include <stdio.h>
int main(){
    int n;
    printf("Input:");
    scanf("%d",&n);
    for( ;n!=0;n=n/10){
        int rem=n%10;
        int rev=0*10+rem;
          printf("%d",rev);
    }
    return 0;
}
